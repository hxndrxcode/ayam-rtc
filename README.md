# ayam-rtc (WebRTC Demo)

Clone this repo:
```
git clone https://github.com/hxndrxcode/ayam-rtc.git
```

Install node modules dependencies
```
npm i
```

Run in development:
```
npm run dev
```

Build static file:
```
npm run build
```

Serve locally with serve
```
npm i -g serve

cd dist

serve
```

Or upload dist directory to static web hosting
